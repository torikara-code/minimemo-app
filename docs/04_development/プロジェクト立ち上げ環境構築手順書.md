# ■ Minimemo 環境構築・Tauriプロジェクト初期化手順書

Tauriを使って「Minimemo」の基盤を構築するためのステップバイステップ手順です。Windows / macOS 両方での開発を想定しています。

---

## Step 1: 前提条件（必要なツールのインストール）

Tauriを動かすためには、バックエンド（Rust）とフロントエンド（Web）のビルド環境が必要です。以下のツールをインストールしてください。

1. **Node.js**
   * [Node.js 公式サイト](https://nodejs.org/) からLTS版をインストールしてください。
2. **Rust (rustup)**
   * [Rust 公式サイト](https://rustup.rs/) の手順に従い、Rustコンパイラとパッケージマネージャ（Cargo）をインストールします。
3. **OS固有のビルドツール**
   * **Windows:** [Visual Studio Build Tools](https://visualstudio.microsoft.com/ja/visual-cpp-build-tools/) をダウンロードし、「C++ によるデスクトップ開発」にチェックを入れてインストールしてください。
   * **macOS:** ターミナルを開き、`xcode-select --install` を実行してコマンドラインツールをインストールしてください。

---

## Step 2: プロジェクトの初期化

ターミナル（またはコマンドプロンプト/PowerShell）を開き、プロジェクトを作成したいフォルダに移動して以下のコマンドを実行します。

```bash
# Tauriアプリの雛形を作成
pnpm create tauri-app --minimemo-app
```

コマンドを実行すると、対話形式でプロジェクトの設定を聞かれます。Minimemoの仕様に合わせて以下のように選択してください。

1. **Project name:** `Minimemo`
2. **Identifier:** `com.minimemo.app`
3. **Choose which language to use for your frontend:** `TypeScript / JavaScript`
4. **Choose your package manager:** `pnpm`
5. **Choose your UI template:** `Solid` (爆速レスポンスのためSolidJSを採用)
6. **Choose your UI flavor:** `TypeScript`

---

## Step 3: プラグインのインストール

Minimemo ではデータ保存に `tauri-plugin-store` v2 を使用します。

```bash
cd minimemo-app
# Rust側とフロントエンド側のプラグインを一括追加
pnpm tauri add store
```

> [!TIP]
> `cargo tauri` がエラーになる場合は、`pnpm tauri` または `npx tauri` を使用してください。また、`pnpm tauri add` を使用すると、Rust側 (`Cargo.toml`) とフロントエンド側 (`package.json`) の両方にプラグインが自動的に追加されます。

---

## Step 4: プロジェクトのセットアップと動作確認

```bash
pnpm install
pnpm tauri dev
```

---

## Step 5: Minimemo仕様へのウィンドウ設定変更

`src-tauri/tauri.conf.json` を開き、透過ウィンドウとプラグインの設定を行います。

```json
{
  "$schema": "https://schema.tauri.app/config/2",
  "productName": "Minimemo",
  "version": "0.1.0",
  "identifier": "com.minimemo.app",
  "build": {
    "beforeDevCommand": "pnpm dev",
    "devUrl": "http://localhost:1420",
    "beforeBuildCommand": "pnpm build",
    "frontendDist": "../dist"
  },
  "app": {
    "windows": [
      {
        "title": "Minimemo",
        "width": 600,
        "height": 260,
        "resizable": false,
        "decorations": false,
        "transparent": true,
        "alwaysOnTop": true,
        "visible": false,
        "center": true
      }
    ],
    "security": {
      "csp": null
    }
  },
  "plugins": {}
}
```

### 【重要】透過ウィンドウを有効にするためのOS設定
macOSの場合、上記に加えてRust側コードでの設定が必要になる場合があります。また、起動時は `visible: false` としているため、`Ctrl+M` を押すまでウィンドウは表示されません。
