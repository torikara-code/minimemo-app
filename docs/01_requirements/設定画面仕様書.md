# ■ Minimemo 設定画面（Settings）詳細仕様書

## 1. 画面表示アプローチ（インアプリ・モーダル）
設定画面は別ウィンドウを開かず、メインウィンドウ内のオーバーレイ（前面レイヤー）として表示する。

* **展開アニメーション:** 設定アイコン（⚙️）クリック時、エディタ画面がさらに強くブラー（ぼかし）で沈み込み、設定パネルがフェードイン＋わずかにスケールアップ（`scale(0.95) -> 1`）して現れる。
* **閉じる操作:** モーダル外枠のクリック、右上の「×」ボタン、または `Esc` キー押下で即座にエディタに戻る。

## 2. 設定値の保存・反映ロジック
* **即時反映（Immediate Reflection）方式を採用する。**
* 「保存ボタン」や「適用ボタン」は配置しない。トグルスイッチやセレクトボックスを変更した瞬間に、SolidJSのSignal（状態）が更新され、UIに即時反映される。
* 同時に裏側で `tauri-plugin-store`（`data.json`）へ非同期保存をかけ、次回起動時にも状態を維持する。

## 3. 設定項目のレイアウトと内容
設定項目はごちゃごちゃさせず、1画面（スクロールなし）に収まるミニマルな構成とする。

| カテゴリ | 項目名 | UIコンポーネント | 選択肢 / 動作 |
| :--- | :--- | :--- | :--- |
| **外観 (Appearance)** | テーマ | セレクトボックス or トグルボタン | Light / Dark / System (OSに同期) |
| **動作 (Behavior)** | 履歴の保持上限 | スライダー (Range Input) ＋ 数値表示 | 10件 〜 200件（デフォルト: 100件） |
| | 表示位置 | セレクトボックス | 画面中央 (Center) / カーソル位置 (Cursor) |
| **システム (System)** | OS起動時に自動実行 | トグルスイッチ (ON/OFF) | ONにするとバックグラウンドで自動待機 |

### 【レイアウトイメージ】
```text
+---------------------------------------------------+
|  Settings                                     [x] |
|                                                   |
|  Appearance                                       |
|  Theme         [ Light | Dark | System ]          |
|                                                   |
|  Behavior                                         |
|  History Limit [----O-------] 100 items           |
|  Window Pos    [ Center ▼ ]                       |
|                                                   |
|  System                                           |
|  Launch at Login                      [ ON (o)]   |
+---------------------------------------------------+
```

## 4. データ構造（`tauri-plugin-store` スキーマ）
JSONファイルとして保存される設定データの構造。

```typescript
type AppSettings = {
  theme: 'light' | 'dark' | 'system';
  historyLimit: number;      // default: 100
  windowPosition: 'center' | 'cursor';
  launchAtLogin: boolean;    // default: true
}
```

## 5. キーボード操作・アクセシビリティ
* 設定ボタンにフォーカスがある状態で `Enter` を押すとモーダルが開く。
* モーダル展開中は、`Tab` キーで設定項目間を移動できる。
* `Esc` キーを押すと、変更内容（すでに即時反映済み）を保ったままモーダルがスッと閉じ、再びエディタにフォーカス（キャレットの点滅）が戻る。思考の邪魔を一切しない。
