# ■ Minimemo アプリ仕様書（v2.0 詳細版）

## 1. プロダクト概要
* **アプリ名:** Minimemo（ミニメモ）
* **対応OS:** Windows / macOS（デスクトップ常駐型）
* **価格モデル:** $15 買い切り（7日間の無料トライアルあり）
* **コンセプト:** **「思考の外部バッファ」**
    * 作業中に浮かんだ思いつき、下書き、断片情報を「0秒」で書き出し、即座に本来の作業に戻れる超軽量テキストツール。
    * キーボード操作のみで完結し、書く体験と閉じる瞬間の快感を最大化する。

## 2. コア体験（UX要件）
1.  **呼び出し・収納の即時性:** グローバルホットキー入力から0.2秒以内で描画完了。
2.  **遅延ゼロのタイピング:** リッチエディタ（WYSIWYG）を使わず、プレーンテキスト入力＋オーバーレイ表示による最速のレスポンス。
3.  **シームレスなキーボード完結:** マウスを使わず、履歴の閲覧・引用・削除・コピーまですべての操作が可能。
4.  **ノイズレスなクリーンUI:** デフォルトを白基調のフロストガラス（LiquidGlass）とし、画面上に常駐させても圧迫感のないデザイン。

---

## 3. システム・技術スタック要件
* **フレームワーク:** Tauri v2.2 + Rust 1.78+（バックエンド）
* **フロントエンド:** SolidJS v1.9 + SolidStart (SPAモード) + TypeScript 5.6
* **データベース:** `tauri-plugin-store` v2 (JSONファイルベースのKey-Valueストア)
* **ショートカット制御:** OSネイティブのグローバルショートカット（Tauri API利用）
* **ウィンドウ制御:** 枠なし（Frameless）、透過（Transparent）、常に最前面（Always on Top / 設定可能）、起動時非表示（バックグラウンド待機）

---

## 4. 画面・UI仕様

### 4.1 メイン入力画面
* **サイズ:** 幅 600px × 高さ 260px（可変設定オプションあり）
* **デザインスタイル (LiquidGlass):**
    * 背景: 半透明の白（ライトテーマデフォルト）、強いブラー効果（`backdrop-filter: blur(24px)`）
    * ボーダー: 微細な白のハイライト（1px solid rgba(255,255,255,0.8)）
    * シャドウ: 深く柔らかなドロップシャドウで浮遊感を演出。フォーカス時はシャドウが微かに青く発光する。
* **構成要素:**
    1.  **設定ボタン (右上):** アイコンのみ。クリックで設定画面展開（テーマ切替、履歴上限などの設定用）。
    2.  **行番号表示 (左端):** 入力行数に同期して自動増減。
    3.  **エディタエリア:** Markdown簡易プレビュー対応（SolidJSによるリアクティブなオーバーレイ方式）。
    4.  **ステータスバー (左下):** リアルタイムの「文字数（chars）」と「行数（lines）」を表示。
    5.  **ショートカットヒント (右下):** 控えめなグレーテキストとバッジで操作案内を表示。

### 4.2 履歴画面（ドロップダウン型）
* **展開位置:** メインウィンドウの下部にシームレスにスライドダウン（高さ最大280px）。
* **リスト要素:**
    * 最新順にカード状で表示。テキストの1行目をプレビュー表示（長過ぎる場合は末尾を省略 `ellipsis`）。
    * 選択中（キーボードフォーカス時）のアイテムは、左端にアクセントカラー（青）の光沢バーが出現。
    * ホバー時または選択時のみ、右端に「×（削除）」ボタンが表示される。

## 5. 機能仕様・操作フロー

### 5.1 キーボードショートカット一覧
| トリガー | 状態 | アクション |
| :--- | :--- | :--- |
| **`Ctrl + M`** | グローバル | メイン入力画面の呼び出し / 格納（トグル） |
| **`Esc`** | メイン画面 | 内容を自動保存して閉じる（空の場合は保存せず閉じる） |
| **`Ctrl + Enter`** | メイン画面 | 内容をクリップボードにコピー ＋ トースト通知 ＋ 閉じる |
| **`Ctrl + K`** | メイン画面 | 履歴画面の展開 / 格納（トグル） |
| **`Arrow Up / Down`**| 履歴展開時 | 履歴アイテムの選択移動 |
| **`Enter`** | 履歴選択時 | 選択した履歴の内容をエディタに復元し、履歴画面を閉じる |
| **`Delete` / `Backspace`**| 履歴選択時 | **選択中の履歴アイテムを削除する**（確認ダイアログなし、アニメーション付きで即削除） |

### 5.2 エディタ機能詳細
* **Markdown 簡易ハイライト (SolidJS 方式):**
    * 入力を受け付けるテキストエリアの背後にハイライト用レイヤー（`div`）を配置。
    * SolidJS の `createSignal` で入力を監視し、正規表現パース結果を即座に反映。
    * **対応記法:** 見出し（`#`）、太字（`**`）、イタリック（`*`）、リスト（`-`または`1.`）、引用（`>`）、インラインコード（`` ` ``）。
* **自動保存機能:**
    * タイピング停止後、またはウィンドウを閉じる瞬間に `tauri-plugin-store` を介してローカルの JSON ファイルへ保存。

### 5.3 履歴管理機能
* **呼び出し:** `Ctrl + K` 押下時、保存済みのデータを取得し、最新順にリスト表示。
* **削除時のUX:** `Delete`キーで削除された際、要素が少し縮みながらフェードアウトするアニメーション（約0.2秒）を挟んでからリストから除外する。即座に次の要素にフォーカスを移す。

## 6. データ構造・保存仕様
* **保存形式:** JSON (tauri-plugin-store)
* **保存パス:** `~/.config/Minimemo/data.json`
* **データモデル (例):**
    ```json
    {
      "memos": [
        {
          "id": "uuid-v4",
          "content": "メモの内容...",
          "updated_at": 1710920000
        }
      ],
      "settings": {
        "theme": "light",
        "history_limit": 100
      }
    }
    ```
* **保持上限:** デフォルト100件（超過時は古いものから自動削除）。

---

## 7. アニメーション・トランジション要件
* **イージング関数:** 思考を邪魔しない「スッと現れ、ピタッと止まる」動き。
    * 推奨 CSS: `transition: all 0.25s cubic-bezier(0.16, 1, 0.3, 1)` (Expo-Out)
* **トースト通知:** `Ctrl + Enter` でコピーした際、画面上部から「Copied to clipboard」の小さなトーストがスライドダウンし、1.5秒後にフェードアウトする。
